Вход в систему

Локальное приложение для расчёта прибыли и закупа товаров по 3 сервисам.

Распределение прибыли по сервисам
Заполните данные по всем сервисам (Excel или вручную) и нажмите «Рассчитать».

Настройки процентов

Только для руководителя. Себестоимость товаров из отчётов автоматически добавляется к строке «Закуп аксессуаров» при расчёте.
